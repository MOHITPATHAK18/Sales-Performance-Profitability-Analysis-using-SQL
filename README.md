# Sales Performance & Profitability Analysis using SQL

## Overview

This project focuses on analyzing retail sales data using SQL to uncover business insights related to sales trends, profitability, customer behavior, return rates, and operational efficiency.

The project demonstrates practical data analysis skills including data cleaning, KPI reporting, advanced SQL querying, and business problem-solving using real-world analytical scenarios.

---

# Objectives

* Analyze sales and profit performance across products and regions
* Identify high-return and loss-making products
* Evaluate customer purchasing behavior
* Measure shipping delays and operational inefficiencies
* Generate KPI-driven reports for business decision-making

# Business Problems Solved

## 1. Sales Trend Analysis

Analyzed monthly and yearly sales trends to identify business growth patterns and seasonal performance.

## 2. Profitability Analysis

Identified high-profit and loss-making products/categories affecting overall business performance.

## 3. Customer Contribution Analysis

Evaluated customer purchasing behavior and contribution to total sales revenue.

## 4. Return Rate Analysis

Analyzed product return trends to identify categories with high return percentages.

## 5. Shipping Delay Analysis

Measured shipping delays and operational inefficiencies impacting customer experience.

## 6. Pareto Analysis

Applied Pareto Principle (80/20 Rule) to identify top customers contributing maximum revenue.
